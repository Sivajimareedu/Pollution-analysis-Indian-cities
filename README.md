# Pollution-analysis-Indian-cities
Pollution Analysis in Indian Cities using R and Data Analytics

## Project Overview
This project analyzes air pollution data from multiple Indian cities between 2015 and 2020 using R. The objective is to identify pollution trends, evaluate Air Quality Index (AQI) patterns, and generate insights through statistical analysis and visualization.

## Technologies Used
- R Programming
- ggplot2
- dplyr
- corrplot

## Dataset
The dataset contains pollution indicators collected from various Indian cities, including:
- PM2.5
- PM10
- NO
- NO2
- NOx
- NH3
- CO
- SO2
- O3
- AQI

## Features
- Data Cleaning and Preprocessing
- Missing Value Handling
- AQI Trend Analysis
- Statistical Analysis
- Linear Regression
- Correlation Analysis
- ANOVA
- K-Means Clustering
- Data Visualization

## Project Workflow
1. Load and preprocess pollution dataset
2. Handle missing values
3. Analyze AQI trends
4. Generate visualizations
5. Perform regression analysis
6. Apply clustering techniques
7. Generate environmental insights

## Results
- Identified pollution trends across Indian cities
- Analyzed AQI variations over time
- Evaluated relationships between pollutants and AQI
- Performed clustering of pollution indicators
